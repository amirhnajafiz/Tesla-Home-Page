Replicating the Tesla home page with Vuetify.

## Start
First install vue-cli:
```shell
npm install -g @vue/cli
```

After that create the project:
```shell
vue create "Name"
```

And then go to project:
```shell
cd "name"
```

Next you need to install vuetify:
```shell
vue install vuetify
```
